# Piqenergy

## Setup

To setup the local environment and run the api:

```bash
cd fastapi
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
fastapi dev main.py
```

To run the sevelte frontend start a new terminal:

```bash
cd piqenergy
yarn install
yarn run dev
```
