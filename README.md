# Piqenergy

## Setup

To setup the local environment and run the api:

```bash
cd fastapi
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
fastapi dev main.py
```

### Start a new terminal

To run the sevelte frontend:

```bash
cd piqenergy
yarm install
yarn run dev
```
